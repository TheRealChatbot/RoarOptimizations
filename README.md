Run start.bat to open the PowerShell GUI.



&nbsp;

&nbsp; GUI Options:

&nbsp; 

&nbsp;   Clean Temp - Deletes recent, temp, prefetch and related clutter to free space and reduce drag.



&nbsp;  

&nbsp;   Disable Telemetry - Attempts to reduce some windows telemetry by stopping known telemetry services and adding host blocks.



&nbsp;   

&nbsp;   Clear Shader Caches - Clears common GPU shader caches so games and apps can rebuild fresh.



&nbsp;  

&nbsp;   Net Refresh - Flushes DNS/ARP and reset Winsock for a clean network stack. Reboot Recommended!



&nbsp;  

&nbsp;   High Performance Plan - Creates and activates a high performance power plan.
