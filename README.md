Run start.bat to open the powershell GUI.

GUI Options:
  
  Clean Temp - Deletes recent, temp, prefetch and related clutter to free space and reduce drag.

  Disable Telemetry - Attempts to reduce some windows telemetry by stopping known telemetry services and adding host blocks.

  Clear Shader Caches - Clears common gpu shader caches so games and apps can rebuild fresh. 

  Net Refresh - Flushes DNS/ARP and reset winsock for a clean network stack. Reboot Recommended!

  High Performance Plan - Creates and activates a high performance power plan.
